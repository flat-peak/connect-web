# FlatPeak Connect-Web

This repository contains the open source FlatPeak Connect-Web App

## Requirements

- [Sign up](https://www.flatpeak.energy/signup) for a free FlatPeak account
- Obtain Publishable Key in the developers section

## Installation

Tools used to develop this project.

- [Node](https://nodejs.dev)

### Run locally

- Clone this repository
- copy `.env.blank` to `.env` and set vars
- Run `npm install --force`
- Run `npm run dev:all`

## Documentation

Complete documentation, instructions, and examples are available at:  [https://docs.flatpeak.energy/docs/connect-web](https://docs.flatpeak.energy/docs/connect-web)
